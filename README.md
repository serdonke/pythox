# Pythox
- An implementation of the [Lox](https://craftinginterpreters.com/the-lox-language.html) language using Python.

Run:

```
python main.py
```

- Can have nicer UX if have uv installed with a better REPL
```
uv run main.py
```
Run Tests:
```
uv run pytest -vv
```
All tests were copied from [craftinginterpreter's test suite](https://github.com/munificent/craftinginterpreters/tree/master/test)
